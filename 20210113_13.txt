13. Да се състави програма, която заделя памет за данни от тип: char, int, long, float, double,
инициализира по време на компилация съответните променливи съответно
в: a=‘a’, b=102, c=50000, d=6, e=-48.907. Програмата да извежда на екрана съдържанието на заделената за данни
памет, форматирано в съответствие с типа на отделните елементи данни.
#include<stdio.h>
int main(){ 
char a='a';
printf("%c\n",a);//a
int b=102;
printf("%d\n",b);//102
int c=50000;
printf("%d\n",c);//50000
float d=6;
printf("%f\n",d);//6.000000
 double e=-48.907;
printf("%lf",e);//-48.907000
return 0;
}
